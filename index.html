TUTORIAL

Super, das klappt doch schon ganz gut! Jetzt darfst du das nächste Quantengatter erforschen und zwar das H-Gatter. Das H-Gatter wirkt wieder auf ein einzelnes Qubit, indem aus den Zuständen 0 und 1 die Superposition erzeugt wird. Wird das H-Gatter auf eine Superposition angewandt, so kann der Zustand 0 entstehen (H0-Gatter) oder der Zustand 1 (H1-Gatter) Bei unserem Spiel wird das H-Gatter ebenfalls gleich auf die gesamte Zeile oder Spalte angewandt.
